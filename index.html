Scala-Skeleton

It's a simple project with sbt tool. For fast start develop in scala. Project contain simple tests and jar building. Licence MIT.

Plugins

* sbt-assembly

* sbt-idea

How to

1. git clone git@github.com:rmuhamedgaliev/scala-skeleton.git
2. mv scala-skeleton $project_name
3. cd $project_name
4. Update project/Build.scala 
5. sbt
6. > test
7. > assemble
8. Enjoy!
